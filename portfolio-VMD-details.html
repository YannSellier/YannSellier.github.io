Project: Votre marché à domicile

  • Category: Webmaster / Web developper
  • Client: ATR2020 home delivery firm
  • Period: juin 2020 - january 2021
  • URL of the project: www.votremarcheadomicile.com

The project details

The body of the votremarcheadomicile.com project was to create a fully functional website that would allow the establishment of a home delivery service for products on the market in several cities of Ile-de-France.

The web site is divided in three parts:

  • Client: which allow the user to see the products and his orders.

  • Merchant: where each merchant can see and edit his products and orders.

  • Administrator: where a supervisor can handle the orders for example to arrange the order to match the best delivery route.



    • One of the major issues was the overwrite of the WooCommerce plugin. Indeed, the site had to offer the possibility to customers to consult and order the products but without having to pay online (because the payment is made on delivery). Thus, to ensure a simple management of products by traders despite their large number (several hundred), I chose to modify the WooCommerce plugin.
    I used its navigation, search and product ordering system but I recreated my own shopping cart page to bypass the payment system. The order processing system then reorganizes and distributes them to merchants and delivery agents.

    Most used tools

  • WordPress
  • WooCommerce with overwrite in php and html/css
  • PhpMyadmin for the database management
  • PlanetHoster as webhost
  • FTP software to remotely edit the files on the server
