Project: C mini projects

  • Category: C programming
  • Period: October 2021

Project details

During the first year of my third year at EPITA, I make many projects in the C language. Here are some of them:

  • My Fnmatch:
    I had to re-implement the function fnmatch function in C.
    fnmatch() compares a single file name against a pattern and returns TRUE if they match else returns FALSE. The comparison is case-sensitive when the operating system uses a case-sensitive file system.

  • Minimake:
    The goal of the Minimake project is to program a simplified version of the classic make(1) from the Unix world.

  • My Malloc:
    The goal was reprogram the malloc function of the C libraries. It was really interesting to learn how the memory management works and how to optimize it.

  • Tiny printf:
    I had to re-implement the function printf function from scratch in C.
    Printf() function is used to print the “character”, string, float, integer, octal, and hexadecimal values onto the output screen. We use printf() function with a %d format specifier to display the value of an integer variable.

  • My Find:
    myfind is a simplified version of find. Finds the element in the given range of numbers. Returns an iterator to the first element in the range [first,last) that compares equal to val. If no such element is found, the function returns last.

  • Lib stream:
    The goal of this project is to implement an file input / output manager from scratch in C. The point of this project is to understand buffering, which is a major I/O performance optimization.
